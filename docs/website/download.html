System Requirements

Minimum and recommended specifications for running SheetAtlas

WIN

Windows

OS: Windows 10 version 1809 or later, Windows 11

RAM: 4GB minimum, 8GB recommended

Disk: 200MB free space

.NET: Included in installer (self-contained)

LIN

Linux

OS: Ubuntu 20.04+, Debian 11+, Fedora 36+

RAM: 4GB minimum, 8GB recommended

Disk: 200MB free space

Dependencies: None (self-contained)

MAC

macOS

RAM: 4GB minimum, 8GB recommended

Disk: 200MB free space

Note: App signing in progress

Installation Guide

Step-by-step instructions for each platform

1

Windows Installation

  1. Download SheetAtlas-Setup-win-x64.exe
  2. Run the installer (may require administrator privileges)
  3. Follow the installation wizard
  4. Launch SheetAtlas from Start Menu or Desktop shortcut
2

Linux Installation (Coming Soon)

Debian/Ubuntu (.deb):

sudo dpkg -i SheetAtlas-linux-x64.deb

Tarball (.tar.gz):

tar -xzf SheetAtlas-linux-x64.tar.gz && ./SheetAtlas
3

macOS Installation (Coming Soon)

  1. Download SheetAtlas-macos-x64.dmg
  2. Open the DMG file
  3. Drag SheetAtlas to Applications folder
  4. Launch from Applications (may require security approval)
